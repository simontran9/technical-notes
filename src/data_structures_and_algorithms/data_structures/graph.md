# Graph

## Overview

| **Operation**                          | **Description**                                                             |
| -------------------------------------- | --------------------------------------------------------------------------- |
| `get_vertices()`                       | Retrieves all the vertices present in the graph.                            |
| `get_edges()`                          | Retrieves all the edges present in the graph.                               |
| `get_neighbors(vertex)`                | Returns a list of neighboring vertices connected to the specified vertex.   |
| `add_vertex(vertex)`                   | Adds a new vertex to the graph.                                             |
| `add_edge(vertex_1, vertex_2, weight)` | Creates a new edge between `vertex_1` and `vertex_2` with the given weight. |
| `contains_vertex(vertex)`              | Checks if the specified vertex exists in the graph.                         |
| `contains_edge(vertex_1, vertex_2)`    | Checks if an edge exists between `vertex_1` and `vertex_2`.                 |
| `remove_vertex(vertex)`                | Removes the specified vertex and all associated edges from the graph.       |
| `remove_edge(vertex_1, vertex_2)`      | Removes the edge between `vertex_1` and `vertex_2` from the graph.          |


also get/set vertex value and edge weight
